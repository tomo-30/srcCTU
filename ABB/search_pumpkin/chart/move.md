# 走行フロー

```mermaid
flowchart TD
    A["走行開始 / 現在位置の初期化"]
    B["現在列の進行方向を決定（+X または -X）"]
    C["直進走行 / 粗スキャン位置まで移動"]
    D{"列の端に到達したか?"}
    E["列端処理 / コーナリング動作"]
    F["隣接列へ移動 (y方向にオフセット)"]
    G["方向を反転し右舷側を畑側に維持"]
    H{"全列を走破したか?"}
    I["走行終了 / 探索完了"]

    A --> B
    B --> C
    C --> D
    D -- "いいえ" --> C
    D -- "はい" --> E
    E --> F
    F --> G
    G --> H
    H -- "いいえ" --> B
    H -- "はい" --> I
